## Data Source

**Dataset:** Lending Club Loan Data  
**Source:** [Kaggle](https://www.kaggle.com/datasets/wordsforthewise/lending-club)  
**Size:** ~2.2M loan records  
**Period:** 2007–2018  

Raw data is not included in this repository due to file size.  
Download the dataset from Kaggle and place `loan.csv` in this folder before running SQL scripts.
